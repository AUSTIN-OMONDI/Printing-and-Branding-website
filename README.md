# Calidad Conquista — Landing Page

Simple static landing page for Calidad Conquista (printing & branding) located in Nairobi, Kenya.

Files added:

- `index.html` — the landing page (hero, services, location, contact form)
- `styles.css` — styling
- `script.js` — wires the contact form to WhatsApp and sets the copyright year

Quick preview

1. Open `e:/LANGGRAPH/calidad_conquista_landing/index.html` in your browser (double-click or right-click -> Open with).
